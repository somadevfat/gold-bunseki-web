# 📊 Gold Volatility Bunseki - 要件定義書 (確定版 v3)

> **プロジェクトの真の目的**
> AI による価格予測（方向当て）はしない。
> 「現在の地合い × 今日控えている経済指標」に対して、**過去の統計データ（事実）** に基づくブレ幅（ボラティリティ）をトレーダーが即座に把握できるダッシュボードを作る。

---

## 1. ほしいもの（機能の核心）

### 1-1. 各地合い（大・中・小）× 経済指標 のボラティリティ
- 特定の指標（例：CPI）を選択したとき、「当時の地合いが大だった回」「中だった回」「小だった回」それぞれの**過去平均ボラティリティ（$）** を表示する。
- 「大地合いのときのCPI発表では平均 $127 動いた」という**事実データ**を出す。
- 予測ではなく、あくまで**過去の統計**（事実の集合体）。

### 1-2. 今日の指標の「前回のチャート」を表示
- 今日 NY_Open に CPI がある場合、**前回のCPI発表日の価格チャート（1分足）** を中央に表示する。
- 数値ではなく**「目で見て波の荒れ狂い具合を直感的に確認する」** ための機能。

### 1-3. 今日の地合い
- 現在どのくらい相場が荒れているか（大・中・小）を表示する。
- 1-1 の「地合いフィルター」の入力値として使う。

---

## 2. 地合い判定ロジック（確定）

### ステップ1: 閾値の自動計算（データドリブン）
- **固定値は使わない。** 毎日 DBの過去1年分のデータから自動計算する。
- 各セッション（Tokyo_AM, London, NY_Open 等）ごとに独立して計算する。
- 計算式：
  ```
  小（Small）: ボラティリティ ≤ 過去1年の33パーセンタイル
  中（Mid）  : 33パーセンタイル < ボラティリティ ≤ 66パーセンタイル
  大（Large）: ボラティリティ > 過去1年の66パーセンタイル
  ```
- こうすることで、金が $2,000 の時代でも $5,000 の時代でも「今の相場環境の中での大中小」に自動追従する。

### ステップ2: 当日の地合い判定
- 直前 **3セッション** のボラティリティをそれぞれ大・中・小に分類する。
- **3つのうち最大（最も荒れた側）の分類を「今日の地合い」として採用する。**
- 理由：「荒れる可能性があるなら大として準備する」= リスク管理の安全策。無理にやることはない。
- 例：
  ```
  直前3コマ → 小, 中, 大 の場合 → 「大」を採用
  直前3コマ → 小, 小, 中 の場合 → 「中」を採用
  ```

## 3. 画面構成

### 【左サイドバー】ナビゲーション & フィルター
- セッション選択（Tokyo_AM, London, NY_Open, NY_Mid 等）
- 地合いフィルター（大・中・小・すべて）で右側の一覧を絞り込む

### 【右パネル】今日のセッション一覧（現在の地合い）
- 各セッションのボラティリティと大中小の分類を一覧表示
- 経済指標がある場合はアラート（重要度付き）を表示
- セッションカードをクリック → 中央に詳細チャートを表示

### 【中央メイン】過去チャートの再現 & 平均ボラ
- **上部**: 選択した「指標 × 地合い」の組み合わせごとの平均ボラ（$）を大きく表示
- **中央**: 前回の同じ指標発表時の1分足チャートを描画（JST基準）
- **下部タブ**: 「前回チャート」「全履歴オーバーレイ」「ボラ比較棒グラフ」の切り替え

## 4. 技術仕様（実装済みと未実装）

### ✅ 完了済み（土台）
1. **データ取得**: XM（MT5）1分足 CSV + 経済指標カレンダー CSV
2. **Pythonエンジン**: XM時間 → JST変換、セッション区分、指標紐づけ
3. **DB（PostgreSQL on Docker）**: セッション別ボラと指標データを格納済み
4. **Go API**: `/api/v1/market/sessions` エンドポイント実装済み

### ❌ 未実装（これから作る）
1. **パーセンタイル計算バッチ**: 毎日DBから大中小の閾値を自動計算して保存
2. **前回チャートAPI**: 指定指標の前回発表日の1分足を返す Go エンドポイント
3. **フロントエンド**: Next.js の画面を本要件定義に合わせて全面改修

---

## 5. 対象データ
- **価格データ**: XAU/USD (GOLD) 1分足 (XM ブローカー、JST換算済み)
- **指標データ**: USD 経済指標カレンダー（重要度: HIGH / MED / LOW）
- **ボラティリティ単位**: **ドル（$）** = セッション内の (最高値 - 最安値)
